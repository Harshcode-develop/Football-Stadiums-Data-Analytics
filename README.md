## Football Stadiums Data Analytics

- This project consists of Data Analytics of Football stadiums by its capacity.
- Extracted data table from wikipedia and cleaned it. 
- Created data pipeline/flow using apache airflow(by using Dags).
- After transforming, data was pushed to azure datalake storage gen2 and validated using Data factory. 
- Used tools and frameworks for data validation and pipeline:
  1. Apache Airflow
  2. Docker
  3. Azure Data Factory
